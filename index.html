HTML Form Reference

Form Tags

Tag Name Description
<form> Form Is used to collect and send information to an external source.
<input> Input Is used to render a variety of input fields on a webpage including text fields, checkboxes, buttons, etc.
<label> Label It provides identification for a specific <input> based on matching values of the <input>'s "id" attribute and the <label>'s "for" attribute.
<textarea> Textarea Is used when creating a text-box for multi-line input (e.g. a comment section). The element supports the "rows" and "cols" attributes which determine the height and width, respectively, of the element.
<select> Select Can be used to create a dropdown list. A list of choices for the dropdown list can be created using one or more <option> elements. By default, only one <option> can be selected at a time.

Form Attributes

Attribute Name Description
required Required In HTML, input fields have an attribute called "required" which specifies that the field must include a value.
maxlength Maxlength In HTML, input fields with type text have an attribute called "maxlength" that specifies the maximum number of characters that can be entered into the field.
minlength Minlength In HTML, an input field of type "text" has an attribute that supports minimum length validation. To check that the input text has a minimum length, add the "minlength" attribute with the character count.
max Max HTML <input>s of type "number" have an attribute called "max" that specifies the maximum value for the input field.
min Min HTML <input>s of type "number" have an attribute called "min" that specifies the minimum value for the input field.
